📋 比赛规则 📋 Tournament Rules

🎾 黄球组 Yellow Ball 🎾 Yellow Ball 黄球组
  • 赛制：一盘定胜负（先到6局获胜）。
  • 无Deuce：金球赛制（40-40时直接决胜分）。
  • 决胜局：若局数达到5-5，第11局进行抢5决胜，最终比分为6-5。
  • Format: Full Set — first to 6 games wins the match.
  • No Deuce: Golden Goal rule applies at 40-40.
  • Deciding Game: If score reaches 5-5, the 11th game is a Tie-break (First to 5); final score 6-5.
🎾 绿翼组 Green Ball 🎾 Green Ball 绿翼组
  • 赛制：三局两胜制，前两局为抢8（先得8分）获胜。
  • 决胜局：若局数1-1，第三局进行抢5决胜。
  • 无Deuce：金球赛制（一分决胜）。
  • Format: Best of 3 Sets — each set is a Tie-break (First to 8 points).
  • Deciding Set: If sets are 1-1, the 3rd set is a Tie-break (First to 5 points).
  • No Deuce: No-Ad Scoring (Golden Point).
🎾 橙星组 Orange Stars 🎾 Orange Stars 橙星组
  • 赛制：三局两胜制，前两局为抢8（先得8分）获胜。
  • 决胜局：若局数1-1，第三局进行抢5决胜。
  • 无Deuce：金球赛制（一分决胜）。
  • Format: Best of 3 Sets — each set is a Tie-break (First to 8 points).
  • Deciding Set: If sets are 1-1, the 3rd set is a Tie-break (First to 5 points).
  • No Deuce: No-Ad Scoring (Golden Point).
📊 排名逻辑 · Ranking Criteria 📊 Ranking Criteria · 排名逻辑
  • 1 胜场 — 胜负场次优先 Wins — Most match wins take priority
  • 2 净胜局 — 赢得局数与失去局数之差 Games Won/Lost Diff — Net games won minus lost
  • 3 胜负关系 — 同分选手间的直接对阵结果 Head-to-Head — Direct match result between tied players
  • ★ 黄球最佳第三名 — 先比较与第1种子的对战结果，若仍相同则比较与第2种子的对战结果 Yellow Ball 3rd Place — Compare results against the 1st seed; if still tied, compare against the 2nd seed
